# DecentraPharma AI 🧬
FOSS decentralized network for AI-driven drug discovery. Volunteers donate compute to train open-source AI models for discovering novel therapeutic molecules.

## Architecture 🏗️
- **Compute Nodes:** Python/Docker workers for cheminformatics tasks.
- **Data Layer:** Public datasets (ChEMBL, PubChem) on IPFS.
- **AI Engine:** PyTorch/DeepChem for molecule generation and binding prediction.

## Quickstart (Run a Node) 🚀
```bash
git clone <your-repo-url>
cd DecentraPharma
pip install -r requirements.txt
python src/node.py
```
